# Homebrew Tap

Homebrew tap for macOS apps and tools by Konrad Michalik.

## Install

```bash
brew tap konradmichalik/tap
```

## Available Formulas

| Formula | Description |
|---------|-------------|
| [`devcap`](https://github.com/konradmichalik/devcap) | Aggregate git commits across repos for standups and time tracking |

### devcap

```bash
brew install konradmichalik/tap/devcap
```
